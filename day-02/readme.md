# Info

Created with [Docker](https://docs.docker.com/engine/install/), [Bun](https://bun.sh/docs/installation) and [TypeScript](https://www.typescriptlang.org/download).

## Install Bun

### Install on Linux

```bash
./init-bun.sh
```

### Install on Windows

```batch
init-bun.bat
```

## Run

### Run on Linux

```bash
./day-02.sh
```

### Run on Windows

```batch
day-02.bat
```
